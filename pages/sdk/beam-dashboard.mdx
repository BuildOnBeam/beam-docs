# Beam Dashboard

<img src="/assets/dashboard/dashboard.png" alt="Beam Dashboard" width="600" />

Beam Dashboard is an all-in-one place for managing your Game and its assets. You can register your game, add contracts, manage funds and API keys.
You can access the game dashboard at:
<Tabs items={["Beta", "Production"]}>
    <Tabs.Tab>
        ```
            https://dashboard.beta.onbeam.com
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```
            https://dashboard.onbeam.com
        ```
    </Tabs.Tab>
</Tabs>

Dashboard allows you to:
- View and manage API Keys for your game
- Manage alerts sent when your BEAM Balance is low
- Manage your game Members (other Developers) and their access to the game
- Manage your games Smart Contracts
- Edit basic game information like name, description, images, PEGI ratings
- Set a webhook for when Players connect to your game
