# API Reference

The Beam API is a RESTful API. Given you have obtained your api key, you are able to connect to the API by using one of our provided server-side SDKs or by communicating through the RESTful endpoints directly over HTTP.

## Specification

If you ever feel the need to access all available endpoints directly from our deployed service, you can use one of the following links:
