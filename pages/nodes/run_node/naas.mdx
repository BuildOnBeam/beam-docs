# Partnered NaaS providers

In case you do not want to operate a node yourself and want a simpler option, we have partnered up with several NaaS (Node-as-a-service) providers which can help you with easy hosting options:

- Luganodes - https://www.luganodes.com/network/?category=upcoming&q=beam
- Zeeve - https://www.zeeve.io/appchains/avalanche-l1s/
- Easeflow - https://docs.easeflow.io/
- NodeOps - https://console.nodeops.network/deployNode
- Gogopool - https://www.gogopool.com/
