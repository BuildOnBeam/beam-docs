# Connecting to Beam
As an EVM-compatible chain, you can connect to Beam network using any Ethereum wallet.

<Callout type="info">
  To access Beam, you need to add the network's network configuration in your desired third-party wallet such as MetaMask. Below we've added the instructions on accessing Beam through MetaMask.
</Callout>

## Connect to Beam Mainnet with MetaMask

| Beam Mainnet   | Settings |
| :----- | :---- |
| Network name    |  `Beam Mainnet`   |
| RPC URL | `https://subnets.avax.network/beam/mainnet/rpc`  |
| Chain ID | `4337`  |
| Currency symbol | `MC`  |
| Block explorer URL| https://subnets.avax.network/beam  |
| One-click setup | <NetworkButton chainId="4337" /> |


<Steps>

### Open MetaMask & Add Network

Launch MetaMask and select the top left option labeled "Ethereum Mainnet". Next, choose "Add Network".

![MetaMask Beam RPC Add Network](/metamask1.png)


### Add a Network Manually

Click on "Add a Network Manually" in the subsequent window.


![MetaMask Beam RPC Add Network](/metamask2.png)

### Input Beam Mainnet Settings
Enter the Beam Mainnet settings as provided above.

![MetaMask Beam RPC Add Network](/metamask3.png)

</Steps>

<Callout type="info">
As an alternative, you can easily integrate the Beam Mainnet settings with a single click through [ChainList](https://chainlist.org/chain/4337). Just click "Add to Metamask" to apply the MetaMask settings.
</Callout>
